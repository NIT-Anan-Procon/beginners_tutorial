次に実数でも計算してみましょう。
演算子は整数の時と同じですが、代入できる変数はdouble型変数である点に注意しましょう。
以下はその一例です。 

class Calculation3 {
    public static void main(String args[]) {
        double real1 = 5.0, real2 = 3.0;
        System.out.println(real1 + real2);
        System.out.println(real1 - real2);
        System.out.println(real1 * real2);
        System.out.println(real1 / real2);
    }
}

出力結果 

8.0
2.0
15.0
1.6666666666666667

このように実数値同士で計算すると演算結果は実数値になります。

では整数と実数とで演算するとどうなるのでしょうか。
以下はその一例です。 

class Calculation3 {
    public static void main(String args[]) {
        int integer = 10
        double real = 1.2;
        System.out.println(integer + real);
    }
}

出力結果 

11.2

このように演算結果はdouble型になります。

なぜこのようになるかというと、
演算は異なる型同士で行った場合表現範囲の大きい方に調整されるという性質があります。

int型は表現できる範囲が整数だけですが、double型は小数点以下の値まで表現できるため、int型よりもdoubleの方が表示範囲が大きいから演算結果はdouble型になったというわけです。

変数の型について説明します。

変数の宣言には型名が必要といいましたが、
変数の型のことを データ型 と言います。

また、intで宣言した変数は「int型」変数と呼ばれます。
このようにデータ型の種類は「○○型」というように呼ぶことが多いです。

数値を表すデータ型は基本的に以下の2つだけです。

型名 数値の種類
int 整数値
double 実数値

int型の変数に入る整数値とは小数点以下を含まない数値です。対してdouble型の変数に入る実数値とは小数点以下を含む数値です。

では実際に使ってみましょう。double型もint型と使い方は同じです。
以下のプログラムはint型変数integerに1をdouble型変数realに1.2を入れたものです。 

class Number {
    public static void main(String args[]) {
        @int integer = 1;
        double real = 1.2;
        System.out.println(integer);
        System.out.println(real);
    }
}

出力結果 

1
1.2

このようにdouble型もint型と同じように使えることがわかったと思います。

では、int型の変数に実数値を入れようとするとどうなるか… 

class Number {
    public static void main(String args[]) {
        @int integer = 1.2;
        System.out.println(integer);
    }
}

このプログラムを実行すると、このようなコンパイルエラーが発生すると思います。 

エラー: 不適合な型: 精度が失われる可能性があるdoubleからintへの変換

このエラーは今後習う演算を行うときに起こりやすいため、注意しましょう。

文字を入れる変数には
文字を1文字だけ入れる変数と文字列を入れる変数の2つがあります。

型名 入れるデータ
char 文字（1文字）
String 文字列

cher型は文字を一つだけ入れることができます。
代入するときは文字を' '(シングルクォーテーション)で囲みましょう。

String型は文字列を入れることができます。
代入するときは" "(ダブルクォーテーション)で囲みましょう。

では実際に使ってみましょう。
以下のプログラムはchar型変数Name1に'A'をString型変数Name2に"Java"を入れたものです。 

class Word {
    public static void main(String args[]) {
        char Name1 = 'A';
        String Name2 = "Java"
        System.out.println(Name1);
        System.out.println(Name2);
    }
}

出力結果 

A
Java

これを使えば同じ文字列を何度も入力しなくてもよくなります。

では実際に計算をしてみましょう。
コンピューターで計算をすることを 演算 と言います。

演算を行うには+や-などの 演算子を使う必要があります。
以下の表に演算子をまとめます。

演算子 行える計算
+ 加算（足し算）
- 減算（引き算）
* 乗算（掛け算）
/ 除算（割り算）
% 除算のあまり

では実際に使ってみましょう。
以下は、ここで紹介した演算子を一通り使ってみた例です。 

class Calculation {
    public static void main(String args[]) {
        System.out.println(10 + 3);
        System.out.println(10 - 3);
        System.out.println(10 * 3);
        System.out.println(10 / 3);
        System.out.println(10 % 3);
    }
}

出力結果 

13
7
30
3
1

10/3(10÷3)の計算結果が 3 になることに注目してください。ここではどの数値も整数として計算しているので、結果も整数になります。
また計算の順序は数学で習ったものと同じです。（乗算除算優先）

変数に計算結果を代入したり、変数同士で計算することも可能です。
以下はその一例です 

class Calculation2 {
    public static void main(String args[]) {
        int number1 = 10 + 3;
        int number2 = 10;
        number2 = number1 + number2
        System.out.println(number1);
        System.out.println(number2);
    }
}

出力結果 

13
23

先程の「number2 = number1 + number2」は
「number2 += number1」と省略することができます。

この代入と演算を合わせた演算子を 代入演算子 と言います。
以下の表に演算子をまとめます。

代入演算子 行える計算
+= a = a + b
-= a = a - b
*= a = a * b
/= a = a / b
%= a = a % b

では実際に使ってみましょう。
以下は、「+=」を使った例です。

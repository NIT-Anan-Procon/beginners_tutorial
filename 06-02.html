1つの条件だけでは不便なので、複数の条件を作りたいときはどうすればいいのか。

else ifというものを使います。実際に見てみましょう。 

class CheckRank {
    public static void main(String args[]) {
        Scanner sc;
        int score;
        score = sc.nextInt();

        if(score >= 80) {
            System.out.println("優");
        } else if(score >= 70) {
            System.out.println("良");
        } else if(score >= 60) {
            System.out.println("可");
        } else {
            System.out.println("不可");
        }
    }
}

「else if」を使うと、「AじゃないけどBのとき」のような処理ができます。

上の例では、もし75点を入力した場合、最初の条件ではじかれて、2番目の条件で引っ掛かります。
2番目の条件文は「70点以上」ですが、実際には初めの条件の影響で「70～79点」ということになります。

もし上の例で、優からさらに分けたいときにどうすればいいでしょう。

そういうときはif文の中にさらにif文を書きます。

これをネストと言って、「Aのとき、さらにBのとき」といった処理ができるようになります。 

class CheckRank {
    public static void main(String args[]) {
        Scanner sc;
        int score;
        score = sc.nextInt();

        if(score >= 80) {
            if(score == 100)
                System.out.println("優：満点です");
            else
                System.out.println("優：惜しい");
        } else if(score >= 70) {
            System.out.println("良");
        } else if(score >= 60) {
            System.out.println("可");
        } else {
            System.out.println("不可");
        }
    }
}

このように同じ条件でも、そのあと別々の処理をすることができます。

今後条件が多くなっていくと「AかつBかつCのとき」といった条件文も出てきます。

先程説明したネストを使ってもいいですが、もっと簡単な方法があります。
それが論理演算子です。

先に種類を説明します。

演算子 意味
&& かつ
|| または
! ～ではない

例えば「AでさらにBのとき」なら 

A && B

「AでもBでもどっちでもいいとき」なら 

A || B

というようにすることで、条件文をまとめることができます。 

!A

のようにすると、「Aではないとき」というふうにすることもできます。

実際に書いてみましょう。もし良だけを判定したいとき。 

class CheckRank {
    public static void main(String args[]) {
        Scanner sc;
        int score;
        score = sc.nextInt();

        if(score >= 70 && score < 80) {
            System.out.println("良");
        }
    }
}

こうすれば、70以上かつ80未満のときだけ良が表示されます。

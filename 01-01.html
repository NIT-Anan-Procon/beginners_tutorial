1年生の皆さん！プログラミング同好会へようこそ！

このサイトは1年生向けに作られたチュートリアルです。このサイトを読み進めていけば、基礎的な プログラミング能力 はもちろん、プログラミング同好会で活動する上で 非常に重要な技術 を学ぶことができます。

ですが、体験入部の人もいるかもしれませんので、初めの方は同好会外でも使えるプログラミングの基礎の基礎しかやりません。ご安心ください。

このサイトの使い方は簡単で、ひたすら上から順に勉強していくだけです。ページ下に 「前のページ」 と 「次のページ」 へのリンクを貼ってあります。右側にも話数と見出しをまとめた 「もくじ」 を用意してあります。前回勉強したところまですぐ飛ぶことができます。

基本的なプログラミングの勉強方法はこの2つです

  • インプット・アウトプット
  • 分からないことを調べる＆人に聞く

まずはインプットです。本やWebサイトを読みすすめてみましょう。

次はアウトプットです。これはかなり重要で、いくらインプットをしたとしても、プログラムを書いていない人のプログラミング能力は絶対に伸びません。まずは どんなものでもいいので、プログラムを書く ところから初めてみましょう！

そして最後のは1番大事なことです。プログラミングを勉強し始めたときは誰でも分からないことだらけです。 

class hoge {
    public static void main(String args[]) {
        System.out.println(s@"Hello World!"@s)
    }
}

↑これは今後皆さんに勉強してもらうプログラミング言語の基本構文です。

もう何が書いてあるか全く分からないですよね？英語と同じで最初はみんな初心者なんです。勉強していく中でなぜ上手くいかないのか、エラーがたくさん出るのか、挫折しそうになることもあるでしょう。そういうときはまず ネットで検索 してみましょう。エラーメッセージをGoogleにコピペするだけでも意外と解決したりすることがあります。同じようなエラーが発生した人が解決策を書いたりしてくれているからです。
それでも分からないときは人に聞いてみましょう。友達でも先輩でもいいです。とにかく1人で悩まず 助けを求める ことが重要です。

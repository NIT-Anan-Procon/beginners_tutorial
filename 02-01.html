ここで教えるやり方はあくまで一例です。もっと便利なやり方もあるのでこの方法が絶対ではないことを念頭に置いてください。

まずはファイルを作りましょう。エクスプローラーの自分のドライブに入ってください。そこに「procon」といった名前のフォルダを作りましょう。そしてその中にもう一度「日付(例：0401)」といったフォルダを作り、入りましょう。
そこで「右クリック→新規作成→テキストドキュメント」で、「日付.java」を作ります。拡張子というのはそのファイルが何のファイルであるかを示しています。

次にテキストエディタについて説明します。皆さんメモ帳というソフトを使ったことはありますか？これもテキストエディタの一種で、簡単に言えばファイルの中身の文字を編集するソフトです。
実はメモ帳でもプログラミングができます。しかし全て黒字でとても見にくいです。 

class hoge {
    public static void main(String args[]) {
        System.out.println("Hello World!");
    }
}

このようにやっぱり色がついてた方が見やすいしかっこいいですよね？そこでプログラマー向けに作られたテキストエディタ「VSCode」 の登場です。（他にも便利な機能があります）

VSCodeは学校のPCに入っているので起動してみてください。自分のPCに入れたい場合は、 環境構築編 へ。

まずはワークスペースにフォルダを追加しましょう。左側のワークスペースというところで「右クリック→ワークスペースにフォルダを追加→指定フォルダを選択」です。そうするとドライブ内のフォルダの中身が反映されます。

次にファイルを編集しましょう。先ほど作ったファイルをクリックしてください。そして下のコードをコピペせずに書き写してください。大きい空白はTabキーです。 

class Printer {
    public static void main(String args[]) {
    }
}

そしてこのファイルを上書き保存しましょう。左上のファイルからしてもいいですが、「Ctrl＋S」の方が早いです。

そして「Ctrl＋@」を押しましょう。これは ターミナル と言って、簡単に言えば コマンド と呼ばれる命令文を打ち込める場所です。プログラムファイルは作るだけではダメで、ちゃんと命令してあげないと動きません。スマホアプリを起動するためにはアイコンをタップしないといけないのと同じです。

ターミナルを出せたらまずしないといけないことは、カレントディレクトリを移動することです。

カレントディレクトリとは今いるフォルダのことです。
プログラムを実行するときは、そのファイルがある場所に移動してから、コマンドを打つ必要があるのです。

ではターミナルの中に以下のコマンドを入力してください。 

cd Y:\procon\日付

cdはChange Directoryの略で、その後ろはディレクトリのパスと呼ばれる、位置を表す文字列になります。まとめると、 Yドライブのprocon/日付の場所に移動 となります。

次に以下のコードを実行しましょう。 

javac 日付.java

この javac というコマンドは、 javaファイルをPCが読めるように翻訳してくれます 。先ほど作ったファイルを翻訳してくれているんですね。この操作を コンパイル といいます。するとフォルダの中にPrinter.classというファイルができたのが分かると思います。これがPCが読めるよう翻訳されたファイルです。

最後に以下のコードを実行しましょう。 

java Printer

この java というコマンドは、さっき出来上がったPrinter.classというファイルを実行してくれています。

流れをまとめると、「ファイル作成→翻訳（コンパイル）→起動（実行）！」といった感じです。特にエラーが出なければ成功しています。何も出てこないのは、まだ中身を書いていないからです。

ではコードの説明をしていきます。 

class Printer {
    public static void main(String args[]) {
    }
}

これは基本的に毎回初めに書くコードで、本来なら説明するべきと言われているところです。しかしそれを説明しだすと頭がパンクする可能性が高いためあえて飛ばします。

言えるのはclassの右にある言葉が クラス名 （そのプログラムの機能の名前とか）で、波括弧で囲われているところがその中身ということだけです。
皆さんが変えるところはそのクラス名だけで大丈夫です。

では、何も出ないのは面白くないので、文字を表示させてみましょう！

2行目の後ろで改行して場所をを空けましょう。その中に以下のコードを打ち込んでください。 

System.out.print("Hello!");

System.out.print は、括弧の中に入っている文字を表示させることができます。そして、文字は必ず 「""（ダブルクォーテーション）」 で囲ってください。これにより文字だよというのを認識させることができます。

最後の「；（セミコロン）」、これは文の終わりを意味しています。日本語で言う「。（句点）」と同じです。

それではコンパイル、実行してみましょう。 

Hello!

Hello!という文字が出ましたか？このようにprintを使えば文字を表示させることができます。

それではこれを 

Hello!
How are you?

のように 2行 で表示したいとします。しかし、printを2行書くだけでは改行はされません。じゃあどうすればいいか。

System.out.printlnというものを使います。これは、自動で末尾に改行を入れてくれます。 

class Printer {
    public static void main(String args[]) {
        System.out.println("Hello!");
        System.out.println("How are you?");
    }
}

こうすると、2行で表示することができます。 

Hello!
How are you?

末尾しか改行できないのは使い勝手が悪い？そうなんです。

なので文の途中で改行したいときは\nを使ってください。 

System.out.println("Hello!\nHow are you?");

改行したいところにこれをそのまま入れることで、改行することができます。

それではコードを見やすくする上で大切なルールを教えます。

この章でとりあえず写してて疑問に思った方もいるかもしれません。なぜTabキーでスペースを空けるのか。

実はこのスペースは無くてもプログラムは動きます。しかし、自分や他の人が見る時にとても不便です。

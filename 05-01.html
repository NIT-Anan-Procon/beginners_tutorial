今まで、決められた値でしかプログラムが動かないのは面白くなかったですよね。

そこで！今回はプログラムを実行した後に入力できるようにしたいと思います。

まずは以下のコードを書いてください。まだ理解する必要はありません。 

import java.util.Scanner

class Printer {
    public static void main(String args[]) {
        Scanner sc;
        String name;
        int age;

        name = sc.next();
        age = sc.nextInt();

        System.out.println(name + "は" + age + "歳です。");
    }
}

一番上のimport java.util.Scannerは入力機能のために必要なものをダウンロードしていると考えてください。

そしてmainの中にScanner sc;を書きます。これは、上でダウンロードしてきたものを使えるようにしていると考えてください。

最後に、実際に入力できるようにしたいときは、 

変数 = sc.next();

のように書きます。こうすることで、入力した文字を指定した変数に入れることができるようになるのです。

ちなみにnextInt()の方は数値専用となります。

実行してみて、初めに名前を、次に年齢を入力してみましょう。

入力内容

太郎
16

出力結果 

太郎は16歳です。

実際には、入力する前の段階で名前を入力してくださいといったように表示しておくと良いでしょう。

「変数」といきなり言われても何のことかわからないと思います。変数とは データを入れる箱 のことです。また、データとは数値や文字列のことです。

これを使えば、計算結果を保存したり、いちいち何度も同じ文字列を書かなくても使い回したりできます。

変数は 自分の好きな名前 を付けます。これによって何を入れたのかわかりやすくなります。

変数の使い方を説明します。

まず、変数を使うためには変数を用意する必要があります。
変数を用意することを 変数の宣言 と言います。
変数の宣言には「型名」と「変数名」を書く必要があります。 

型名 変数名;

型名とは変数が入れられるデータを決めるものです。
今は整数値に対応するintだけを覚えていてください。
（詳しくは「変数の種類」を参照）

変数名とはそのままの意味で、変数の名前のことです。
変数の使い方にはいくつかルールがあります。

  • 名前に使える文字は英数字及びアンダーバー
  • 数字は一文字目に使用不可
  • 大文字と小文字は別物（区別される）
  • 予約語は使用不可（例:public ,intなど）

また、「,」を使うことで複数の変数を同時に宣言することも可能です。 

  型名 変数名1, 変数名2;

変数が使えるようになったので、変数に記憶させてみましょう。

変数にデータを記憶させることを 代入 と言い、仕方は以下の通りです。 

変数名 = 代入するデータ;

この「＝」は数学の等号とは違う意味です。
「＝」は 代入演算子 と言い右側のデータを左側の変数に記憶させるという意味です。


また、宣言と代入は一度に行えます。
この宣言と代入を同時に行うことを 「変数を宣言と同時に初期化する」 といいます。 

型名 変数名 = 代入するデータ;

これも宣言と同じように複数の変数をまとめて初期化ができます。 

型名 変数名1 = データ, 変数名2 = データ;

では実際に書いてみましょう。
次のコードは、intで宣言した変数Numberに10の値を代入しています。 

class Variable {
    public static void main(String args[]) {
        @int Nunber;
        Number = 10;
        System.out.println(Number);
    }
}

出力結果 

10

また、以下のように変数を初期化すると、より短く書けます。
この場合でも先ほどと同じ結果が得られます。 

class Variable {
    public static void main(String args[]) {
        @int Nunber = 10;
        System.out.println(Number);
    }
}

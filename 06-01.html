毎回決められた処理をするのは面白くないので、今回はif文について説明したいと思います。

if文とは、ifの意味で分かるように、「もし～ならAの処理、そうじゃないならBの処理」というふうに場合によって処理を分けることができるのです。

実際に書いてみましょう。 

import java.util.Scanner

class CheckAdult {
    public static void main(String args[]) {
        Scanner sc;
        int age;
        age = sc.nextInt();

        if(age < 18)
            System.out.println("18歳未満");
        else
            System.out.println("18歳以上");
    }
}

ifの括弧の中に書いてあるのは条件式と呼ばれ、正しいか間違っているかを判断する式です。

そして、その下に処理を書くことで条件分岐を作ることができます。

今回であれば「ageの中に入っている値が18よりも大きいとき18歳未満と表示」、「そうでないとき18歳以上と表示」という分かれ方をしています。

先ほど使った「<(小なり)」という記号、これは比較演算子の一種となります。基本的には数学と同じで、

演算子 意味
> より大きい
< より小さい
>= 以上
<= 以下
== 等しい
!= 等しくない

となり、等しいのときにイコールを2個使うのに注意です。

普通のif文では処理は1つしかできません。それでは2つ以上の処理をしたいときはどうすればいいでしょうか。

それは 波括弧を使う ことで解決できます。今まで書いてきたclassの波括弧のように、その中に入っているということを示すために使うことができ、if文や今後やる他の制御文も2つ以上の処理を書くことができます。 

import java.util.Scanner

class CheckAdult {
    public static void main(String args[]) {
        Scanner sc;
        int age;
        age = sc.nextInt();

        if(age < 18) {
            System.out.println("18歳未満");
            System.out.println("未成年です");
        } else {
            System.out.println("18歳以上");
            System.out.println("成年です");
        }
    }
}

波括弧を使って、1つの処理だけしか書かないということも可能です。

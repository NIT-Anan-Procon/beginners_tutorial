次に文字列と演算子についでです。

文字列は数値ではないため計算はできませんが、文字列の結合は「+」の演算子でできます。以下はその一例です。 

class Puls {
    public static void main(String args[]) {
        String str1 = "情報";
        String str2 = "コース";
        String str3 = str1 + str2;
        System.out.println(str3);
    }
}

出力結果 

情報コース

また、文字列と数値を「+」の演算子でつなげると数値は文字列に変換されて結合されます。以下はその一例です。 

class Puls {
    public static void main(String args[]) {
        String str1 = "No." + 2;
        String str2 = 1 + 2 + ":" + 1 + 4;
        System.out.println(str1);
        System.out.println(str2);
    }
}

出力結果 

No.2
3:14

str2が少しややこしいので説明すると、
最初の「1 + 2」は数値として計算され3となります。その後は「":"」が続くので3は文字列に変換されて":"が連結されます。続く「1 + 4」は3:が文字列であるため 1と4は文字列に変換され連結します。
